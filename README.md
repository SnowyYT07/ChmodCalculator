# Chmod Calculator

``` shell 
user@localhost:$ echo "Easy Chmod Calculator in C" > info.txt
```

<h3 align="center"><b>How to use it</b></h3>

- `git clone https://github.com/Snowyyt07/ChmodCalculator`
- `cd ChmodCalculator`
- `gcc ChmodCalculator.c -o chmodcalculator`
- `./chmodcalculator`
